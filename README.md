产品介绍
「秒著」是一款基于AI的软件著作权申请材料自动生成工具。用户只需上传项目文档（.doc/.docx/.pdf），系统通过AI智能提取关键信息，自动生成符合国家版权局要求的「软件概述」和「设计说明书」两份核心申请材料，并导出为标准格式的Word文档。

核心价值
🚀 效率提升：将传统需要1-3天的软著材料撰写工作缩短至几分钟
📝 降低门槛：无需专业知识产权知识，人人都能生成符合规范的申请材料
🎯 标准输出：生成的文档完全符合官方规范，一次通过
🖼️ 智能配图：自动从上传文档中提取图片并智能嵌入对应章节
功能特性
Step 1 - 配置 API
支持多家主流API提供商（火山引擎、DeepSeek、智谱、Kimi、MiniMax）
API密钥本地持久化存储，安全便捷
一键测试API连接
智能布局：API提供商、API Key、测试连接按钮一行显示
Step 2 - 上传文件
支持点击上传 / 拖拽上传
支持多文件上传
支持格式：.doc, .docx, .pdf
单文件大小限制：10MB
备选：直接粘贴文档文本内容
自动从docx文档中提取嵌入图片
Step 3 - 确认信息
AI自动提取17+个核心字段
支持手动编辑修改
高级参数折叠展示，带自动填充默认值
Grid两列布局，页面高度控制在1-2屏以内
响应式设计，小屏幕自动切换单列
Step 4 - 生成结果
分两步生成：软件概述 + 设计说明书
Tab切换预览两份文档
一键复制到剪贴板
下载为.docx格式
Markdown表格自动转换为Word表格
图片智能匹配到对应章节
技术栈
组件	技术选型
前端	单HTML文件（原生JS）

快速开始
环境要求
Node.js 14+
npm 6+
安装运行
# 1. 克隆或下载项目
cd 软著生成器

# 2. 安装依赖
npm install

# 3. 启动服务
node proxy.js

# 4. 打开浏览器访问
http://localhost:3000
快速启动脚本
Windows: 双击 启动秒著.bat
macOS: 双击 启动秒著.command
使用指南
完整操作流程
启动代理 (node proxy.js)
    ↓
打开浏览器 (localhost:3000)
    ↓
Step 1: 选择API提供商 → 输入API Key → 保存
    ↓
Step 2: 上传文档(.doc/.docx/.pdf) 或 粘贴文本
    ↓
    [前端解析文档 + 提取图片]
    ↓
Step 3: AI提取信息 → 用户核对/修改
    ↓
Step 4: AI生成两份文档 → 预览 → 下载.docx
生成文档规范
软件概述.docx
封面：软件名称 + “软件概述” + 公司名 + 日期
正文：
信息表（17个字段）
开发目的
面向领域/行业
软件的主要功能（表格）
软件的技术特点
产品架构图
页眉：软件名称 + 页码（第X页 共X页）
表格样式：标题行浅灰色背景，深黑色边框
设计说明书.docx
封面：软件名称 + “设计说明书” + 公司名 + 日期
目录页：自动生成
正文章节：
项目背景
项目目标（功能架构图 + 功能目标表）
项目范围
项目时间表
需求描述及界面展示
图片嵌入：智能匹配到对应章节
页眉：软件名称 + 页码
界面预览
Step 1 - 配置 API
选择API提供商并输入API密钥，一键测试连接。

Step 2 - 上传文件
点击或拖拽上传文档，支持多选。
